## :book: 文档说明

每个章节通常包含如下文件，你可以使用 `_template_vuln` 文件夹来创建新章节：

- README.md —— 漏洞描述、利用方法及常用Payload
- Intruder —— 可直接用于Burp Intruder的文件集
- Images —— 用于README.md的配图
- Files —— README.md中引用的其他文件

你可能还会喜欢AllTheThings系列的其他项目：

- [InternalAllTheThings](https://swisskyrepo.github.io/InternalAllTheThings/) —— Active Directory及内网渗透备忘清单
- [HardwareAllTheThings](https://swisskyrepo.github.io/HardwareAllTheThings/) —— 硬件/IOT渗透测试Wiki

还想了解更多？可查看[书籍推荐](https://github.com/swisskyrepo/PayloadsAllTheThings/blob/master/_LEARNING_AND_SOCIALS/BOOKS.md)和[Youtube频道精选](https://github.com/swisskyrepo/PayloadsAllTheThings/blob/master/_LEARNING_AND_SOCIALS/YOUTUBE.md)。
